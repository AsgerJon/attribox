AttriBox: Advanced Attribute Management for Python

AttriBox is a sophisticated Python framework designed to enhance attribute management within classes, providing a controlled environment that prevents common pitfalls such as the unintentional sharing of mutable objects across instances. Inspired by challenges encountered with signal-slot mechanisms in PySide6, AttriBox introduces a set of tools that make instance-specific attribute management clearer, safer, and more intuitive.

Features

  • Singleton Placeholders: Utilizes singleton classes like this and scope to reference current and owning instances, ensuring clarity and intentionality in attribute handling.
  • Typed and Delayed Descriptors: Implements advanced descriptor classes, enabling deferred instantiation and strict type-checking for attributes, fostering safer and more predictable class behaviors.
  • Customizable Attribute Boxes: Offers AttriBox, a versatile descriptor for managing complex attribute instantiation patterns, providing syntactic sugar for class definitions and enhancing readability.
  • Meta-programming Support: Leverages metaclasses and abstract base classes to enforce framework conventions and facilitate elegant solutions to common object-oriented design challenges.

Quick Start

Here's a quick example of how to use AttriBox to create a class with managed attributes:

python
from attribox import AttriBox, this

class ExampleInner:
    def __init__(self, owner, *args, **kwargs):
        self.owner = owner

class CustomClass:
    my_attr = AttriBox[ExampleInner](this)

    def __init__(self, name):
        self.name = name

# Using the custom class
custom_instance = CustomClass("MyInstance")
print(custom_instance.my_attr)
